### 🎉 初始功能

#### 1. 批量报备

- ✅ 支持多商户 ID 同时报备
- ✅ 空格/换行分隔多个商户 ID
- ✅ 单个失败不影响其他商户

**示例：**
```
输入：104890 104888 103084 报备
输出：
  ✅ 104890 → 主扫 H5 切换成功
  ✅ 104888 → 主扫 H5 切换成功
  ✅ 103084 → 主扫 H5 切换成功
```

---

#### 2. 401 自动刷新 Token

**机制：**
```javascript
async request(url, data, accessToken) {
  try {
    return await axios({...});
  } catch (err) {
    if (err.response?.status === 401) {
      const newToken = await this.getNewToken();
      // 重试请求
    }
  }
}
```

**优势：**
- ✅ Token 永不过期（自动刷新）
- ✅ 无需手动更新代码
- ✅ 减少因 Token 失效导致的报备失败
