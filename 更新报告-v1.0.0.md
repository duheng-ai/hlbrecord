# 批量火脸主扫报备技能 v1.0.0 更新报告

**发布日期**: 2026-04-13

---

## 🎉 核心功能

### 1. 批量报备

支持一次报备多个商户 ID：

```
输入：104890 104888 103084 报备

输出：
📋 批量报备完成（共 3 个）：

✅ 104890 → 主扫 H5 切换成功
✅ 104888 → 主扫 H5 切换成功
✅ 103084 → 主扫 H5 切换成功
```

---

### 2. 401 自动刷新 Token

Token 失效时自动重新登录，无需手动处理。

---

### 3. 独立容错

单个商户失败不影响其他商户：

```
输入：104890 999999 报备

输出：
✅ 104890 → 主扫 H5 切换成功
❌ 999999 → 主扫 H5 切换失败：商户不存在
```

---

## 📝 使用方法

### 触发词

```
<商户 ID1> <商户 ID2> ... 报备
```

### 示例

```
104890 报备
104890 104888 报备
104890 104888 103084 报备
```
